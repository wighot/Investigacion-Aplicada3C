Operador Spread


El operador spread permite que un elemento iterable como un arreglo o una cadena pueda ser extendido, para entender mejor el operador spread nos permite expandir o distribuir los elementos de un array u objeto en las partes del código donde se admitan esos elementos, es decir, imaginemos que tenemos que tenemos un array que contiene 5 números y otro array que contiene 6 números, a través de este operador spread y sus sintaxis podemos unir estos dos array teniendo como resultado un nuevo array con 11 números, también con esto podemos copiar, añadir elementos y unir array.

Su sintaxis

Su sintaxis en simple y no es difícil de entender, con sus sintaxis podemos hacer lo siguiente:

➤ Para llamar una función


-Por ejemplo tenemos nuestra función y queremos llamar a una array para esa función su sintaxis es la siguiente:

” FuncionEjemplo(…ArrayEj)“

➤ Para unir dos arreglos


-Por ejemplo, si tenemos dos arreglos y necesitamos que se unan solo hacemos lo siguiente:

” var = ejemplo = [ …arreglo1, …arreglo2];“

➤ Para objetos literales o cadena de caracteres:


-Simplemente es ejecutar nuestro arreglo o adicional incluir información predefinida, por ejemplo

”[ …arreglo1, …arreglo2, ‘4’, ‘cuatro’ ];“
