Funciones flecha


Como ocurre con todos los avances tecnológicos de hoy en día, Javascript es uno más en una larga lista de lenguajes ( en este caso, de secuencia de comandos) que ha pasado por diferentes versiones, cada una mejorando la anterior y abriendo puertas que hacen posible una nueva forma de codificar las aplicaciones para las que se utilice este lenguaje interpretado, en 2015 se implementa la versión

Con la introducción de ECMAScript 6 se hace posible el uso de una nueva forma de declarar funciones en js, este metodo es conocido como “funciones flecha”. Dicho en otras palabas, es una alternativa compacta a la expresión tradicional de una función, más no se puede usar en otras situaciones o tareas en js.



Su sintaxis

➤ Un parámetro con una expresión simple no requiere del “return”.



➤ Varios parámetros necesitan estar dentro del parénteris, pero si se trata de una expresión simple no se requiere del “return”.



➤ Las declaraciones de varias líneas requieren el uso de las llaves y “return”.
